_Last updated: July 10, 2026_

MeshTrax ("the App") is a free, open-source client for MeshCore LoRa mesh
networking devices. By installing or using the App, you agree to these Terms of

## 1. The App

MeshTrax is an independent, community-built client. It requires compatible
MeshCore hardware to function. The App lets you exchange messages with other
participants over a **decentralized LoRa mesh network**. Messages travel
peer-to-peer between radios; there is no MeshTrax server that stores, hosts, or
relays your content.

There is **zero tolerance** for objectionable content and abusive behavior.

- Any message can be **reported** to the developer (from the message's menu),

Because the network is decentralized, the developer does not monitor, host, or
control the content that other participants transmit. The developer will,
